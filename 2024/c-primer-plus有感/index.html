enum DAY{
    MON = 1, TUE, WED, THU, FRI, SAT, SUN
}
enum可以为每个常量指定一个整数值，如果没有指定，那么默认从 0 开始递增。

对scanf和printf函数的部分转换说明

转换说明 含义
%e、%f、%g、%a 解释为浮点数
%i、%d 解释成有符号十进制整数
%o 解释成有符号八进制整数
%u 解释成无符号十进制整数
